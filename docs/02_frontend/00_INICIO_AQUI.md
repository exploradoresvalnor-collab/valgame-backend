## üîó ENDPOINTS PRINCIPALES

### Auth
- `POST /api/auth/register` - Registro
- `POST /api/auth/login` - Login
- `POST /api/auth/forgot-password` - Recuperar
- `POST /api/auth/refresh-token` - Refrescar token

### Personajes
- `GET /api/characters` - Obtener personajes
- `POST /api/characters` - Crear personaje
- `GET /api/characters/:id` - Detalles
- `PUT /api/characters/:id` - Actualizar

### Items/Equipamiento
- `GET /api/items` - Listar items
- `GET /api/items/:id` - Detalles
- `POST /api/inventory/equip` - Equipar
- `POST /api/inventory/unequip` - Desequipar

### Marketplace
- `GET /api/marketplace/listings` - Ver listados
- `POST /api/marketplace/list` - Crear listado
- `POST /api/marketplace/buy/:listingId` - Comprar
- `POST /api/marketplace/cancel/:listingId` - Cancelar

### Tienda/Compras
- `GET /api/shop/packages` - Ver paquetes
- `POST /api/purchases/initiate-stripe` - Compra Stripe
- `POST /api/purchases/verify-web3` - Compra Web3

### Combate
- `POST /api/combat/start` - Iniciar combate
- `POST /api/combat/attack` - Atacar
- `GET /api/combat/status` - Estado combate

### Salud
- `GET /api/health` - Health check
- `GET /api/health/ready` - Readiness probe
- `GET /api/health/live` - Liveness probe

**Ver `00_BACKEND_API_REFERENCE.md` para detalles completos**
